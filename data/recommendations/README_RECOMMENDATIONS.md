## 10. **Common Questions Answered**

**Q: Where do I put my recommendation data?**
A: In `index.html`, find `const recommendationsData = [` around line 680

**Q: Do I need to edit the documentation files?**
A: No, they're just for reference. Only edit `index.html`

**Q: Can I add more than 3 recommendations?**
A: Yes, just copy the object structure and increment the ID (200, 201, 202...)

**Q: Which file shows code examples?**
A: `COPY_PASTE_READY.md` Part 3, 10 and `RECOMMENDATIONS_TEMPLATE.js`

**Q: Where do I get avatar images?**
A: See `RECOMMENDATIONS_SETUP.md` Step 3 (3 free options provided)

**Q: How do I test locally?**
A: See `COPY_PASTE_READY.md` Part 8 (detailed testing instructions)

**Q: What if something breaks?**
A: See `COPY_PASTE_READY.md` Part 7 (troubleshooting table)

**Q: Can I change the section title?**
A: Yes, see `COPY_PASTE_READY.md` Part 6 or `RECOMMENDATIONS_SUMMARY.md`

**Q: How many recommendations should I have?**
A: 3-5 is ideal. More can slow the page.

A: No, you can use 3, 4, or 5. But 5 is more meaningful.
